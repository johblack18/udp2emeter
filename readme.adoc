= loxone UDP to SMA Energy Meter converter
Gerhard Sonnleitner

== Introduction

This sketch may be used to read udp telegrams from loxone, convert it to SMA energy-meter telegrams and send it via UDP.





As hardware you need:

* an ESP8266 (tested with NodeMCUv2 and WeMos D1 mini)
